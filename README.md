# Web-Based Calculator

A simple, easy-to-use web-based calculator with a clean interface, advanced features, and a persistent theme. It supports basic arithmetic operations, as well as more complex calculations like square roots and percentages. The calculator also keeps a history of your calculations for easy reference.

## Features

-   **Basic Arithmetic**: Perform addition, subtraction, multiplication, and division.
-   **Advanced Functions**: Calculate square roots and percentages.
-   **Calculation History**: View a history of your past calculations.
-   **Theme Switching**: Toggle between light and dark themes to suit your preference.
-   **Responsive Design**: The calculator is designed to work on a variety of screen sizes.
-   **Keyboard Support**: Use your keyboard to enter numbers and perform calculations.

## How to Use

1.  **Open `index.html`**: Open the `index.html` file in your web browser to start using the calculator.
2.  **Perform Calculations**:
    -   Click the number buttons or use your keyboard to input numbers.
    -   Use the operator buttons (`+`, `-`, `*`, `/`) for arithmetic operations.
    -   Press the `=` button or the `Enter` key to see the result.
3.  **Advanced Functions**:
    -   **Square Root**: Enter a number and click the `√` button.
    -   **Percentage**: Enter a number and click the `%` button to convert it to a percentage.
4.  **Clear**:
    -   Click the `C` button to clear the current input.
5.  **History**:
    -   View your calculation history on the right-hand side.
    -   Click the **Clear** button in the history panel to remove all previous calculations.
6.  **Theme Switching**:
    -   Click the **Toggle Theme** button to switch between light and dark modes. Your preference is saved for your next visit.

## Project Structure

The project is organized into three main files:

-   **`index.html`**: The main HTML file that defines the structure of the calculator, including the display, buttons, and history panel.
-   **`style.css`**: The stylesheet that controls the appearance of the calculator. It includes styles for both the light and dark themes.
-   **`script.js`**: The JavaScript file that implements the calculator's logic. It handles user input, performs calculations, manages the history, and controls theme switching.
